# 📝 PWA Checklist Diário

Um aplicativo de lista de tarefas simples, intuitivo e instalável (PWA), feito para ajudar na organização das atividades do dia a dia.

## 🚀 Demonstração

O projeto está publicado e pode ser acessado pelo link abaixo:
👉 **[Acesse o Checklist aqui](https://hadjars.github.io/PWA-checklist/)**

---

## ✨ Funcionalidades

* **Adicionar Tarefas:** Digite sua tarefa e adicione à lista.
* **Marcar como Concluído:** Clique em um item para riscar (marcar como feito).
* **Limpar Lista:** Botão para remover todas as tarefas de uma vez.
* **PWA (Progressive Web App):** Pode ser instalado no celular ou computador e funciona como um aplicativo nativo.
* **Responsivo:** Layout adaptado para celulares e desktops.
* **Persistência de Dados:** Suas tarefas ficam salvas no navegador mesmo se você fechar a aba (LocalStorage).

---

## 📲 Como "Baixar" e Instalar

Como este é um **PWA**, você não precisa ir até a Play Store ou App Store. Siga estes passos:

### No Android (Chrome):
1. Acesse o [link do projeto](https://hadjars.github.io/PWA-checklist/).
2. Clique nos **três pontinhos** no canto superior direito.
3. Selecione **"Instalar aplicativo"** ou **"Adicionar à tela inicial"**.
4. O ícone aparecerá na sua lista de apps!

### No iOS (Safari):
1. Acesse o [link do projeto](https://hadjars.github.io/PWA-checklist/).
2. Clique no botão de **Compartilhar** (ícone do quadrado com uma seta para cima).
3. Role para baixo e clique em **"Adicionar à Tela de Início"**.

### No Computador (Chrome/Edge):
1. Acesse o link pelo navegador.
2. Na barra de endereços (URL), clique no ícone de **instalação** (computador com uma seta) ao lado da estrela de favoritos.
